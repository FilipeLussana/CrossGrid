Objetivo
-------
Arquivos com prompts prontos para enviar ao Codex (OpenAI Codex / Copilot) para implementar as próximas atividades do projeto `CrossGrid`.

Como usar
---------
1. Abra o prompt desejado (ex.: `implement_rules.md`).
2. Cole o texto na interface do Codex / Copilot ou use a API da OpenAI para enviar como instrução de código.
3. Revise as alterações sugeridas pelo modelo e aplique-as ao repositório (commit/PR).

Observações
-----------
- Cada prompt inclui contexto do repositório e arquivos-alvo. Ajuste caminhos se necessário.
- Teste manualmente cada implementação antes de marcar como concluída.
